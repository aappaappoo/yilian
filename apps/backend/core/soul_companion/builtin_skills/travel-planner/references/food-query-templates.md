# Food Query Templates

本文件用于构造美食 `local_search` 查询。

## 基本参数

- `place`：优先用已选主景点、商圈、车站或午休区域，例如“中山路”“厦门科技馆”“厦门站”。
- `category`：`food`。
- `keyword`：结合用餐时间、老人偏好、预算和忌口。

## 模板

老人午餐：
`小吃` 或 `清淡`

不吃海鲜：
`清淡` 或 `小吃`

预算低：
`小吃` 或 `平价`

靠近车站：
`小吃` 或 `简餐`

## 查询纪律

- 不要在景点或主要活动区域确定前泛查城市级美食。
- 高德 POI 更适合短关键词，不要把“午餐、老字号、有座位、适合老人”等条件全部塞进一个长关键词。
- 如果餐厅结果中有可用于行程的候选，优先筛选，不要为了更地道、更有名继续查询。
- 只有结果为空、全部离主景点/车站明显太远，或全部与饮食偏好冲突时，才补查餐厅。
