# Weather Routing

本文件用于天气返回后决定景点方向。

## 不适合户外的天气

如果天气包含大雨、雷雨、阵雨、台风、大风、暴晒、高温、寒潮、明显降温等，景点查询必须转向：
- 室内。
- 有遮挡。
- 少走路。
- 打车方便。
- 靠近车站、商圈或午休点。

此时不要推荐出海、海钓、沙滩久待、长距离海边步道、爬山或需要长时间排队的户外项目。

如果用户喜欢海钓、出海、看海，但天气不适合出海：
- 说明天气限制。
- 改查或推荐海景替代点、室内展馆、骑楼老街、商圈休息点。

## 厦门示例

雨天、老人、一日游时，景点查询优先围绕：
- 厦门科技馆。
- 厦门市博物馆。
- 中山路骑楼/中华城商圈。
- 鹭江道/轮渡附近短停。

高德 POI 检索更适合具体 POI 或短关键词。不要把“室内、少走路、打车方便、适合老人”等筛选条件全部塞进 `keyword`，这些条件由你在拿到结果后筛选和解释。

雨天厦门老人一日游的首查建议：
- `place=厦门`
- `category=scenic`
- `keyword=厦门市博物馆 厦门科技馆 文化馆`

如果首查为空或明显不可用，再换一个具体 POI，例如 `厦门市博物馆` 或 `中山路`。不要连续用多个复合关键词试探。

## 天气适合户外

如果天气适合户外：
- 可考虑看海、老街、轻松步道、近车站景点。
- 仍要控制步行强度，老人场景不要安排高强度路线。
